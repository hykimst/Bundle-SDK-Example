# Bundle SDK Example
## Step 1
1) Add your credentials in `.env` file

## Step 2 
1) download the latest Bundle SDK from https://matterport.github.io/showcase-sdk/sdkbundle_home.html#latest-version.
2) Unzip the folder
3) rename from `showcase-bundle` to `bundle`
4) Move the `bundle` folder to the root folder of this project.

## Step 2
1) npm install
2) npm start
